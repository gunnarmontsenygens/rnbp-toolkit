# Restricted N-Body Problem in Jacobi Coordinates Toolbox (MATLAB)

## Summary

This folder contains MATLAB utilities for modeling, simulating, and analyzing the **Restricted N-Body Problem (RNBP)** using **Jacobi coordinates**.

The toolbox extends the **N-Body Problem in Jacobi Coordinates Toolbox** by introducing a **massless particle** whose motion is governed by the gravitational field generated by the massive bodies. It provides routines for propagating the restricted equations of motion, transforming between barycentric and Jacobi coordinate systems, reconstructing barycentric trajectories, evaluating dynamical quantities, and visualizing the resulting motion.

The implementation follows the Jacobi-coordinate formulation developed in **ASEN 6062 – Celestial Mechanics**, allowing the translational motion of the massive-body barycenter to be removed while expressing the equations of motion in reduced coordinates.

Throughout the toolbox, the gravitational constant is normalized to

- `G = 1`

unless otherwise stated.

## Functions

### Mass Utilities

### m2M.m

**Description:** Computes the cumulative mass sequence associated with an N-body system.

The routine converts the individual body masses into the cumulative masses

```text
M_j = Σ_{i=1}^{j} m_i
```

required by the Jacobi-coordinate formulation.

#### Inputs

| Variable | Description | Units |
|---------|-------------|-------|
| `m_list` | Individual body masses | – |

#### Output

| Variable | Description | Units |
|---------|-------------|-------|
| `M_list` | Cumulative mass sequence | – |

#### Notes

- Used throughout all Jacobi-coordinate transformations.

## Notes

- The equations of motion are formulated using **Jacobi coordinates**, eliminating the translational motion of the barycenter of the massive bodies.
- The final body is assumed to have **zero mass** and therefore does not influence the motion of the primaries.
- Throughout the toolbox, the gravitational constant is normalized to **G = 1**.
- Coordinate transformation routines allow conversion between **barycentric** and **Jacobi** coordinate systems.
- Conserved quantities of the massive-body N-body system can be monitored to assess numerical integration accuracy.
- Unless otherwise stated, all routines assume an inertial reference frame.

## Vector Format Conventions

- Position and velocity vectors are stored as row vectors,

```text
r_i = [x_i  y_i  z_i]

v_i = [vx_i  vy_i  vz_i]
```

- Position and velocity lists follow the conventions

  - `r_vec_list` → `N × 3`
  - `v_vec_list` → `N × 3`

  where each row corresponds to one body.

- The Jacobi position vectors are stored as

```text
R_vec_list = [R₁;
              R₂;
              ⋮
              Rₙ₋₁]
```

- The Jacobi velocity vectors are stored analogously.

- The complete Jacobi state vector is ordered as

```text
Xjac = [R₁;
        V₁;
        R₂;
        V₂;
        ⋮
        Rₙ₋₁;
        Vₙ₋₁]
```

- The complete barycentric state vector is ordered as

```text
Xbar = [r₁;
        v₁;
        r₂;
        v₂;
        ⋮
        rₙ;
        vₙ]
```
